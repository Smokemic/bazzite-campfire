## 🏕️ What is Bazzite Campfire?

Bazzite Campfire is a **safe, interactive learning environment** for anyone who wants to switch from Windows to Linux – specifically to **[Bazzite](https://bazzite.gg/)**.

Everything runs in your browser. Nothing touches your real system. No credit card. No login. Just learning.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🖥️ **Safe Terminal** | Try Linux commands like `ls`, `pwd`, `whoami` – no risk |
| 🤖 **Bazzi Chat** | Ask about Spotify, Discord alternatives (Fluxer!), gaming |
| 🎬 **Campfire Cinema** | Embedded YouTube tutorials for each command |
| 🎒 **Backpack** | Collect badges for completed missions |
| 🏕️ **4 Learning Missions** | First Steps · Files & Folders · Exploring · Gaming |
| 🔥 **3D Campfire** | Animated fire, sparks, and low-poly Bazzi robot |
| 🌍 **Multi-language ready** | English, German, Turkish, Spanish, French |
| 🚫 **No BS** | No credit card, no login, no tracking, open source |

## 🗺️ Missions

| Mission | Commands | What you'll learn |
|---------|----------|-------------------|
| **1: First Steps** | `ls`, `pwd`, `whoami` | Basic terminal orientation |
| **2: Files & Folders** | `ls -l`, `ls -a`, `cd` | Navigating and viewing files |
| **3: Exploring** | `date`, `cal`, `echo` | System information |
| **4: Gaming on Linux** | `which steam`, `flatpak list`, `lutris` | How gaming works on Linux |
